## 二、快速安装

### 1、clone到本地

`git clone git@github.com:baozouai/rn-cartoon-app.git`

### 2、打开终端，`cd rn-cartoon-app`,切换到项目根目录，在项目根目录运行以下命令

`npm install`，安装所需的依赖
`react-native run-ios` 运行项目
